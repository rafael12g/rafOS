6. NAVIGATION FLOW EXAMPLE
   ------------------------

   User journey:
   1. Launch browser: `browser`
   2. View home page (default)
   3. Press [2] → Navigate to About
   4. Press [3] → Navigate to Documentation  
   5. Press [B] → Go back to About
   6. Press [B] → Go back to Home
   7. Press [4] → Navigate to Search
   8. Press [1] → Return to Home
   9. Press [Q] → Quit browser

   History stack automatically manages up to 10 pages!


7. FEATURES SUMMARY
   ----------------

   ✓ 4 complete web pages with rich content
   ✓ Navigation between pages using number keys
   ✓ Back button with automatic history management
   ✓ Refresh functionality to reload current page
   ✓ Clean, intuitive UI with URL bar
   ✓ Custom rafos:// protocol
   ✓ Fully functional in text mode (80x25)
   ✓ Integrated help in main RafOS help command
   ✓ Exit to shell at any time with [Q]


8. TECHNICAL SPECS
   ----------------

   Language:        x86 Assembly (16-bit)
   Code Size:       ~300 lines
   Total Size:      ~2KB (code + pages)
   Pages:           4 built-in pages
   History Depth:   10 pages maximum
   Display Mode:    Text mode 80x25
   Integration:     Direct kernel integration
